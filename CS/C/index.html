    3.项目¶

    一个源代码文件太长，适合分成几个文件，but 两个独立的源代码文件不能编译形成可执行的文件
    项目→包含多个源代码文件 （dev c++编译：把一个项目中所有的代码文件都编译后，链接起来
    而在 VS（Visual Studio）,编译按钮是对单个源代码文件编译，构建按钮是对整个项目做链接
    一个.c是一个编译单元

    4.基本数据类型¶

    4.1 原码、补码、反码¶

    正数：符号位为0，原码补码反码均相同

    原码：符号位为1，其余各位表示绝对值
    反码：符号位为1，其余各位为原码取反
    补码（将正数与负数的计算统一起来）：反码+1

    -1的二进制码是全1（所有Bit都是1）
    最大的无符号整数的二进制也是全1（所有Bit都是1）

    Text Only
    int a=-1;  
unsigned b=10;  
if(a<b){   
    printf("<");   
}else{   
    printf(">");    
}

    输出：>
    带符号整数会在比较之前转换成相同类型的无符号整数

    C语言中，32位的int
    最大值是2^31 -1
    最小值是- 2^31
